# Content: Ultrathinker Personal Brand

Content management for the anonymous "Ultrathinker" founder persona.

## Folders

- `drafts/` - Content awaiting approval
- `published/` - Posted content with engagement data
- `ideas/` - Topic backlog and future content ideas

## Privacy

CRITICAL: Never include:
- Real name
- Product name (vyx)
- Identifying implementation details
- Screenshots with identifying info

See `.claude/skills/founder-content/SKILL.md` for full guidelines.

## Workflow

1. Run `/content-review` for daily content suggestions (pulls from vyx repo)
2. Review and approve/edit drafts
3. Post via agent (Steel MCP) or manually
4. Move to `published/` with engagement notes
